# Retail Sales Power BI Project

This repository contains a dirty retail sales dataset created for practicing
real-world data cleaning, transformation, and dashboard building in Power BI.

## Dataset Details
- 10,000 retail sales records
- Indian customer names and cities
- Inconsistent text formatting (uppercase/lowercase, extra spaces)
- Missing values
- Inconsistent date formats
- Broken KPI columns for recalculation

## Purpose
This dataset is intentionally messy to simulate real business data.
It is ideal for:
- Power Query data cleaning
- Data transformation
- KPI creation
- Retail sales dashboard development in Power BI

## Tools Used
- Power BI
- Power Query
- GitHub
